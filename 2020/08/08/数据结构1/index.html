title: 数据结构（1）
##线性表##
###定义：线性表是具有相同的数据类型的n(n>=0)个数据元素的有限序列，其中n为表长，当n为0时线性表是一个空表。

###特点：表中元素有限；表中元素具有逻辑上的顺序性，表中元素有其先后次序；表中元素是数据元素；表中元素的数据类型都是相同的，即每个元素占有相同大小的存储空间；表中元素具有抽象性，即仅讨论元素逻辑关系，不考虑元素究竟是什么内容。

##1.顺序存储##

###定义：线性表的顺序存储又称顺序表。

###特点：表中元素的逻辑顺序与其物理顺序相同

###线性表的顺序存储类型描述为：

  • 静态分配

      #define MaxSize 50         //定义线性表的最大长度  
  type struct{ 
  ElemType data[MaxSize];   //顺序表的元素
  int length;               //顺序表当前长度
  }Sqlist                      //顺序表定义类型

  • 动态分配

      #define MaxSize 100                      //表长度的初始定义  
  type struct{ 
  ElemType *data;                       //指示动态分配数组的指针
  int MaxSize，length;               //数组最大容量和当前个数
  }Sqlist                                //动态分配数组顺序表类型的定义

  C语言初始动态分配的语句为：
  L.data=(Elemtype*)malloc(ElemType)*InitSize);

  C++:
  L.data=new Elemtype[InitSize];
